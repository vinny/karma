# Karma System for phpBB

[![Tests](https://github.com/vinny/karma/actions/workflows/tests.yml/badge.svg)](https://github.com/vinny/karma/actions/workflows/tests.yml)

This extension adds a voting system to your phpBB forum, allowing users to upvote or downvote posts, similar to Reddit.

## Features

- **Upvote and Downvote**
- **Member Rankings**
- **Moderator Tools**
- **Notifications**
- **Forums Exclusions**
